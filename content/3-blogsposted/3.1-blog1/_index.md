## AWS and the orientation to build AI Agent for businesses

AWS wants to become the main platform to help businesses build, control and operate **AI agents** at production scale.

At **AWS Summit New York 2026**, Swami Sivasubramanian, AWS Vice President of Agentic AI, presented many important announcements related to AI agents, AI security, software modernization and data for automated workflows.

---

## 1. Main announcements at AWS Summit New York 2026

### Amazon Bedrock AgentCore has many new capabilities

AWS introduced new features that help AI agents connect with internal enterprise knowledge, the web, and paid data sources.

The goal is to help businesses:

- Build agents faster.
- Control agents more effectively.
- Continuously improve agents when they are deployed to production.

---

### Amazon Bedrock Managed Knowledge Base

Amazon Bedrock Managed Knowledge Base makes it easier for businesses to build **RAG** ​​pipelines.

This service supports:

- Data connector.
- Handles many document formats.
- Smart retrieval mechanism for multi-step queries.

---

### Web Search on Amazon Bedrock AgentCore

The Web Search feature allows AI agents to search for new information from the web in a pre-managed way.

Agents can:

- Get the latest information from the internet.
- Cite sources clearly.
- Keep data in AWS customers' secure environment.

---

### AWS WAF supports monetization of AI bot traffic

AWS WAF adds the ability for content publishers to control traffic from AI bots or AI agents.

Publishers can:

- Content or API access pricing.
- Measure AI bot traffic.
- Collect fees when bots or agents access content.

---

### Bedrock AgentCore Harness is generally available

Bedrock AgentCore Harness helps developers run production-grade AI agents without having to write all the orchestration logic themselves.

Developers can configure:

- Model.
- Tool.
- Skill.
- Instruction.

---

### AWS Continuum and Security Agent

AWS introduces security tools that use AI to speed up the process of detecting and addressing security risks.

Key capabilities include:

- Prioritize vulnerabilities according to business impact.
- Verify exploitability.
- Supports threat modeling according to the STRIDE model.
- Scan pull requests.
- Suggest bug fixes right in the developer's workflow.

---

### Kiro cho iOS

Kiro has a native iOS application, allowing developers to work with sessions right on the phone.

Developers can:

- Initialize session.
- Monitor progress.
- Session navigation.
- Xem diff.
- Browse changes from mobile devices.

---

### AWS DevOps Agent has release management capabilities

AWS DevOps Agent has added release management capabilities.

Agents can:

- Assess readiness level before release.
- Run automated tests.
- Testing in a production-like environment.

---

### AWS Transform Continuous Modernization

AWS Transform supports continuous repository analysis to detect and handle technical debt.

This tool can:

- Continuous codebase analysis.
- Detect technical debt.
- Prioritize issues that need to be handled.
- Create your own pull request to fix errors.

---

### Amazon S3 Annotations

Amazon S3 Annotations enable rich metadata and context to be attached directly to objects.

This is useful for:

- AI agent.
- Automated workflow.
- Systems need to understand data at large scale.

---

## 2. The problem is raised

Businesses want to apply AI agents but still encounter many barriers:

- Data is scattered in many systems, making it difficult to input into AI accurately.
- AI agents need new information from the web but must still ensure confidentiality and cite the source.
- As agents become more powerful, risks of control, access and governance also increase.
- Current software security is still slow compared to the speed of development and release.
- Technical debt accumulates in the codebase, but analyzing and fixing bugs often takes weeks.
- Digital content is accessed by AI bots, but it is difficult for content owners to control or collect fees.
- Data metadata for AI often has to be managed by a separate system, causing complexity.

---

## 3. Solution offered by AWS

AWS offers a family of solutions around AI automation, including:

- **Bedrock AgentCore**: Build and operate controlled AI agents.
- **Managed Knowledge Base + Web Search**: Helps agents access reliable web and business knowledge.
- **AWS WAF AI Traffic Monetization**: Helps content owners control and collect fees for AI bots.
- **AWS Continuum / Security Agent**: Detect, prioritize, and support security fixes faster.
- **DevOps Agent**: Release review and pre-production testing.
- **AWS Transform**: Automatically detect and reduce technical debt.
- **S3 Annotations**: Adds context to data serving AI agents and automated workflows.

---

## 4. Conclusion

Announcements at AWS Summit New York 2026 show that AWS is focusing strongly on bringing AI agents into real enterprise environments.

Not just stopping at creating simple chatbots or AI assistants, AWS wants to provide a full ecosystem so businesses can:

- Connect agents with internal data.
- Allow agents to access the web securely.
- Access management and governance.
- Automate software security.
- Reduce technical debt.
- Optimize release process.
- Manage data metadata at large scale.

This shows that AI agents are gradually becoming an important part of modern enterprise technology infrastructure.
