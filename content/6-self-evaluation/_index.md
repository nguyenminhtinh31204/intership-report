After the internship, I achieved several key results:

- Learned how to use AWS Management Console to create and manage resources.
- Understood the role of services such as S3, Lambda, API Gateway, DynamoDB, Cognito, Amplify Hosting, and CloudWatch.
- Participated in building the **Serverless AI Invoice Scanner** system.
- Took responsibility for backend APIs such as upload, invoice retrieval, search, and invoice update functions.
- Learned how to test APIs with Postman.
- Learned how to use CloudWatch Logs to inspect and debug Lambda errors.
- Learned how to write technical documentation and weekly worklogs.
- Gained more experience working in a team on a real cloud project.

## Areas for Improvement

Besides the results achieved, I still need to improve in several areas:

- Improve discipline and strictly follow the rules of a company or organization.
- Improve time management to complete tasks on schedule.
- Improve problem-solving thinking, especially when a system has many connected services.
- Communicate better in daily work, present ideas more clearly, and handle situations more professionally.
- Improve English technical reading skills.
- Study AWS security more deeply, especially IAM, Cognito Authorizer, Secrets Manager, and API security.
- Practice writing cleaner, more maintainable, and better-structured code.

---

## Lessons Learned

Through the internship and project work, I learned several important lessons:

- Theory should go together with practice to understand how a system really works.
- When working with cloud systems, security, access permissions, and resource costs must be considered carefully.
- API Gateway, Lambda, S3, and DynamoDB are important components in serverless architecture.
- CloudWatch Logs are essential for debugging during development.
- Teamwork requires clear task division and regular communication.
- Technical documentation makes deployment, testing, and project handover easier.
- AWS resources should be checked and cleaned up after a lab to avoid unnecessary costs.

---

## Overall Evaluation

Overall, I evaluate that I completed the internship well. I learned new knowledge about AWS Cloud, better understood serverless architecture, and had the opportunity to apply that knowledge to a real project.

Although I still have limitations in discipline, communication, and problem-solving thinking, I made progress throughout the learning, implementation, and project completion process. This was an important experience that gave me a stronger foundation to continue developing in cloud computing and backend programming.
