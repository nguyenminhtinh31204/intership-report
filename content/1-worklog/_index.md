This work diary summarizes my internship progress during the **12-week AWS internship program**. The internship was divided into two main phases: learning AWS fundamentals through **First Cloud Journey** and applying the knowledge to build the project **Serverless AI Invoice Scanner**.

During the first phase, I focused on studying cloud computing concepts, AWS core services, serverless architecture, authentication, deployment, and monitoring. In the second phase, I applied those skills to design, implement, test, deploy, document, and finalize a serverless invoice processing system using AWS services and AI integration.

The internship was completed in **12 weeks**, with the weekly worklog shown below:

## Main Project

The main project developed during the internship was **Serverless AI Invoice Scanner**.

This project is a serverless web application that allows users to upload invoice files, process invoice content using AI, store extracted invoice data, and manage invoice records through a web interface.

The project uses the following services:

- AWS Amplify Hosting
- Amazon Cognito
- Amazon API Gateway
- AWS Lambda
- Amazon S3
- Amazon Textract
- OpenAI API
- Amazon DynamoDB
- Amazon CloudWatch
- AWS IAM
